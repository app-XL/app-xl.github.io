muApps

Lego Blocks for enterprise apps

One of the key outcome of the muServLabs is muApps which is a lego modeled app assembly framework following microservices architecture pattern. muApps contains various lego modules (from AuthN/Z to key plumbing modules to functional blocks like workflow, orchestration, rules etc.,) build as polyglots which can be quickly assembled using simple json configuration components to shape up an enterprise class digital application addressing the factors like - AutoScale, Autotune - DevOps based continuous delivery with Docker based containerization - Service discovery and orchestration with built in appfabric - Etcâ€¦

Repos
  • muApps-Main: Lego module bootstrap code generator to kickstart an app module
  • muApps-Core: App core components and framework libraries
  • muApps-Data: Generalized data modeller and controller for polyglots DBs
  • muApps-Fabric:
  • muApps-AuthN: oAuth based AuthN components
  • muApps-AuthZ: Rolebased AuthZ components
  • muApps-Wf: Event action based business workflow engine with operational blocks to quickly implement actions
  • muApps-Actions: An action pack all standard action which can be plugged or trigger for business actions
