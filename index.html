Explore, Evolve, Enhance Micro Service Patterns

Community actively work on building software engineering patterns, block and components on various principles and designs evolved by microservice architecture community http://microservices.io/. The objective of the community is to augment the think process of the microservices.io community and build experimental and PoC components or reusable frameworks to facilitate the rapid design of microservice architecture based solutions.

muApps

muApps is a lego modeled app assembly framework following microservices architecture pattern. muApps contains various lego modules (from AuthN/Z to key plumbing modules to functional blocks like workflow, orchestration, rules etc.,) build as polyglots which can be quickly assembled using simple json configuration components to shape up an enterprise class digital application addressing the factors like - AutoScale, Autotune - DevOps based continuous delivery with Docker based containerization - Service discovery and orchestration with built in appfabric - Etcâ€¦
